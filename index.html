Interactive Exercises

These exercises represent standardized client content from Expand Interactive, where I worked as a front end developer. Designs were created as a team, at the whiteboard or in the browser.

All exercises were written with proprietary autoplay audio. This has been removed at the request of the content owner.

All drag and drop interactions occur only in one direction for better voice command accessibility.

Text-on-screen contrast passes WCAG AA and AAA guidelines.

All UI elements that use color for feedback also appropriately use text or iconography for visual reinforcement.

  • Button Choice

    This is a binary choice exercise with clickable images for user input.

    Button Choice exercise

  • Order Sort

    This sorting exercise features draggable responses and a “check your answers” button, allowing the user to try multiple times if needed.

    Order Sort exercise

  • True False

    This true/false exercise uses a swipe-left/swipe-right mechanism below the mobile breakpoint, and uses more of the container width at larger sizes.
